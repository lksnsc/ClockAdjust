# ClockAdjust

Ferramenta simples para sincronizar o relógio do Windows.

## Como usar

1. Execute o `ClockAdjust.exe` como administrador  
2. Clique em **Corrigir agora**

## Avisos

- Requer permissões de administrador  
- Pode haver alerta de antivírus (falso positivo devido ao empacotamento)  
- Depende do serviço de horário do Windows estar ativo
